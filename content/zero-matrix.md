### Zero Matrix
The [matrix](matrix.md) $0_{mn}$ is the matrix of size $m \times n$ filled only with zeros. If using multiple digit numbers, a comma must be used between them.
$$
0_{23}=
\begin{bmatrix}
	0 & 0 & 0 \\
	0 & 0 & 0 \\
\end{bmatrix}
$$
