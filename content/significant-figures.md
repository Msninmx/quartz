### Significant Figures
If there's no decimal point, start counting from the right and skip zeroes.
If there's a decimal point, count from the left and skip zeros.

##### Example:
| Number    | Significant Figures |
| --------- | ------------------- |
| 200       | 1                   |
| 3500      | 2                   |
| 700,000   | 1                   |
| 20,001    | 5                   |
| 6,200,000 | 2                   |
| 801,300   | 4                   |
| 200.      | 3                   |
| 200.0     | 4                   |
| 3.050     | 4                   |
| 0.007     | 1                   |
| 0.0020    | 2                   |
| 1.0020    | 5                   |
